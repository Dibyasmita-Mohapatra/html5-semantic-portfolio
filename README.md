# Html5 Semantic Portfolio

A modern, accessible, and semantic multi-page personal portfolio website built using **HTML5** and **CSS3**. This project demonstrates best practices in semantic HTML, accessibility (WCAG), SEO optimization, and responsive web design.

---

## 📌 Project Overview

This portfolio website was developed as part of a **Web Development Assessment**.

The primary objective is to build a portfolio that follows modern HTML5 semantic standards and Web Content Accessibility Guidelines (WCAG), ensuring that the website is accessible to all users while maintaining excellent SEO performance.

---

## 🚀 Features

* Semantic HTML5 structure
* Responsive layout
* WCAG accessibility best practices
* Keyboard-friendly navigation
* Skip-to-content link
* Accessible contact form
* ARIA labels where appropriate
* SEO-friendly meta tags
* Open Graph metadata
* Canonical URLs
* Mobile-friendly design
* Clean and modern UI
* Lighthouse optimized
